# NOTICE

## Purpose of This Repository

This repository serves as a **public showcase / portfolio presentation** of the **Hullah** graduation project, completed as part of the **CPCS 499 Senior Project** at the Department of Computer Science, Faculty of Computing and Information Technology, King Abdulaziz University (May 2025).

It is intended to:

- Demonstrate the **scope, design, methodology, and outcomes** of the project to recruiters, reviewers, and collaborators.
- Provide a high-level technical overview without exposing implementation details.

## What This Repository Does **Not** Include

To protect the integrity and privacy of the project, this repository **does not contain**:

- Source code (frontend, backend, AI model implementation, or scripts)
- Datasets or any training/testing data
- Database schemas, database files, or database credentials
- API keys, secrets, environment configuration files, or deployment configs
- The full graduation project report (unless explicitly published by the authors)
- Any user data, screenshots containing personal information, or proprietary assets

## Content Included

This repository includes only:

- A professional project overview (`README.md`)
- Summarized public documentation describing the project's purpose, features, architecture, methodology, and results (`docs/`)

All content has been written based on the publicly presentable parts of the original graduation report, summarized and rewritten for a general audience.
